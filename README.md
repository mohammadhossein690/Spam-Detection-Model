# Spam Detection
This project implements a text classification model to automatically detect and filter spam messages using the **Naive Bayes** algorithm.

## Key Highlights
- **Technique:** Natural Language Processing (NLP) with Vectorization.
- **Model:** Naive Bayes Classifier.
- **Goal:** Improving communication security by filtering unwanted content.

## Methodology
- **Preprocessing:** Cleaning text data (tokenization, removing stopwords).
- **Feature Extraction:** Converting text to numerical data using **CountVectorizer** or **TF-IDF**.
- **Modeling:** Training a Naive Bayes model for binary classification (Spam vs. Ham).

## Key Results
- **Model:** Naive Bayes
- **Accuracy:** 0.979
- **Key Metrics:** Confusion Matrix, Precision, and Recall.
